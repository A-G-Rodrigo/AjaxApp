Bienvenidos a esta prueba de Ajax

¡ATENCIÓN! Esto es una aplicación demo a la que acceden varios usuarios. Esto quiere decir que todos los datos, documentos o imágenes que usted añada o modifique, podrán ser visualizados, modificados o borrados por cualquier otro usuario activo. La función de esta demo es mostrar el funcionamiento de la aplicación con datos genéricos o de ejemplo y nunca deben tomarse como datos reales.

Esta Aplicación Web está desarrollada íntegramente por Álvaro García Rodrigo, inspirada en CloudGestión.
